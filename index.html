Pokemon API Documentation

Base URL:

https://pokemonapilab220240312105041.azurewebsites.net

Endpoints:

Get All Pokemons

GET /pokemons

Description: Retrieves a list of all pokemons.

Response:

[
    {
        "id": 1,
        "name": "Squirtle",
        "type": "water"
    }
]

Get a Pokemon by ID

GET /pokemon/:id

Description: Retrieves a pokemon by its ID.

Parameters:

  • id (integer) - The ID of the pokemon.

Response:

{
    "id": 1,
    "name": "Squirtle",
    "type": "water"
}

Add a New Pokemon

POST /pokemon/:name/:type/:id

Description: Adds a new pokemon to the database.

Parameters:

  • name (string) - The name of the pokemon.
  • type (string) - The type of the pokemon.
  • id (integer) - The ID of the pokemon.

Response:

{
    "id": 2,
    "name": "Bulbasaur",
    "type": "grass"
}

Update a Pokemon

PUT /pokemon/:id

Description: Updates an existing pokemon.

Parameters:

  • id (integer) - The ID of the pokemon.

Request Body:

{
    "name": "Squirtle",
    "type": "water"
}

Response:

{
    "id": 1,
    "name": "Squirtle",
    "type": "water"
}

Delete a Pokemon

DELETE /pokemon/:id

Description: Deletes a pokemon.

Parameters:

  • id (integer) - The ID of the pokemon.

Response:

{
    "message": "Pokemon deleted successfully."
}

Error Responses:

  • 400 Bad Request: Invalid parameters or request body.
  • 404 Not Found: The requested pokemon does not exist.
  • 500 Internal Server Error: Server encountered an unexpected condition.

Note:

  • Replace :id with the actual ID of the pokemon when making requests.
  • Ensure proper error handling for all endpoints.
